# Titanic Data Analysis with Pandas

This project performs data analysis using the Titanic dataset.

## Part 1
A custom dataset was created using a Python dictionary and converted into a Pandas DataFrame.

Requirements:
- 5 columns
- 15 rows
- Custom index

## Part 2
Analysis of the Titanic dataset including:

1. Data Exploration
   - head()
   - info()
   - describe()

2. Data Cleaning
   - Missing Age → replaced with median
   - Missing Embarked → replaced with mode
   - Cabin column removed
   - Duplicates removed

3. Data Analysis
   - Survival rate by gender
   - Survival rate by class
   - Average age per class
   - Survival rate by age group

4. Filtering
   - Female survivors
   - Surviving children
   - Surviving 1st class passengers

5. Insights
   - Women were more likely to survive
   - Higher class passengers survived more
   - Children were prioritized
   - Female passengers in 1st class had the highest survival rate
